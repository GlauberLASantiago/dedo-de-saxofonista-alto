Guia para o Usuário

Saxofonista, assim que o app é aberto, ele já está programado para: tocar em um andamento (BPM) mais lento; iniciar com uma contagem de 2 compassos; estar com o metrônomo ligado; tocar em loop; utilizar som de saxofone alto; exibir o guia visual das chaves para o primeiro pentagrama do sistema; e reproduzir todos os instrumentos. O saxofone alto é um instrumento transposto em Eb — a afinação padrão já está configurada corretamente.

Para visualizar melhor a partitura, clique no botão "Foco".

Para abrir uma nova música: Selecione um arquivo de partitura (.mxl, .musicxml) no botão de pasta, ou utilize o menu de hinos para carregar partituras diretamente da biblioteca.
